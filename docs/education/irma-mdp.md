# IRMA – Management Development Program (Project Management for Rural Livelihoods)

**Type:** Short-term MDP  
**Year:** 2021

**Focus:** Project lifecycle structuring, monitoring frameworks, resource planning, outcome tracking.

**Relevance:** Reinforced disciplined execution & metrics thinking later applied to engineering pipeline observability and iterative delivery.

## Condensed Syllabus Outline
### Project Structuring
??? note "Lifecycle Frameworks"
    - Initiation → Planning → Execution → Monitoring → Closure

??? note "Scope & WBS"
    - Work Breakdown Structures, dependency mapping

### Monitoring & Evaluation
??? note "Indicator Design"
    - Output vs outcome vs impact levels

??? note "Data Collection & Quality"
    - Field data reliability, validation loops

### Resource & Risk
??? note "Resource Planning"
    - Allocation matrices, utilization tracking

??? note "Risk Management"
    - Identification, qualitative/quant assessment, mitigation plans

### Stakeholder & Reporting
??? note "Communication Cadence"
    - Dashboards, status artifacts, narrative framing

### Practical Workshop Components
- Case simulations (project recovery, schedule slippage)
- Mini exercise: KPI selection & monitoring storyboard

## Technical Transfer
Structured milestone tracking parallels pipeline stage instrumentation & alert design in data engineering contexts.
