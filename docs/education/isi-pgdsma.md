### Probability & Inference
??? note "Probability Theory"
    - Axioms, combinatorics, conditional probability, law of total probability & Bayes
    - Random variables (discrete/continuous), expectation operators
    - Important distributions (Binomial, Poisson, Normal, Exponential, Gamma, Beta)

??? note "Estimation Theory"
    - Point estimators: bias, variance, MSE, MVUE concepts
    - Method of Moments, Maximum Likelihood
    - Sufficient statistics (Fisher–Neyman factorization intuition)

??? note "Hypothesis Testing"
    - Neyman–Pearson framework, Type I/II errors, power
    - z / t / chi-square / F tests
    - p-values vs confidence intervals linkage

### Regression & GLMs
??? note "Linear Regression"
    - OLS derivation (normal equations) & geometric interpretation
    - Diagnostics: residual analysis, influence (Cook’s D), multicollinearity (VIF)

??? note "Regularization & Extensions"
    - Ridge, Lasso, bias–variance interplay
    - Logistic regression (odds, logit, likelihood)

### Multivariate Analysis
??? note "Exploratory Multivariate"
    - Mean vector & covariance matrix, correlation structure
    - PCA: eigen decomposition, variance explained, loading interpretation

??? note "Classification & Grouping"
    - Linear / Quadratic Discriminant Analysis assumptions
    - Clustering basics: k-means objective, hierarchical linkage concepts

### Time Series
??? note "Foundations"
    - Stationarity, ACF/PACF interpretation
    - AR, MA, ARMA, ARIMA identification & parameter estimation

??? note "Forecasting & Evaluation"
    - Residual diagnostics (Ljung–Box), information criteria (AIC/BIC)
    - Error metrics (MAE, RMSE, MAPE) selection context

### Optimization (Intro)
??? note "Linear Programming"
    - Problem formulation, feasible region geometry
    - Simplex method intuition

??? note "Unconstrained Basics"
    - Gradient / Hessian, convexity cues
