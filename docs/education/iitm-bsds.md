# IIT Madras – BS Data Science & Applications (Ongoing)

**Program Type:** Bachelor of Science (Online, Flexible Mode)

**Focus Areas:** Foundations of Data Science, Statistics, Machine Learning, Computational Thinking.

**Highlights:**
- Structured learning path blending programming, statistics, ML.
- Emphasis on scalable analytical thinking and applied problem solving.

## Overview
Blended multi-term program emphasizing rigorous mathematical & computational foundations with practical application labs. The modular architecture (Foundational → Diploma → Degree) enables progressive specialization while ensuring depth in probability, statistics, ML workflows and data systems.

## Detailed Syllabus (Progressively Updated)

### Foundations
??? note "Computational Thinking"
    - Problem decomposition, abstraction, pattern recognition
    - Pseudocode design, control structures, recursion basics
    - Algorithmic complexity (intro), iterative refinement

??? note "Mathematics for Data Science"
    - Sets, functions, sequences, series
    - Basic calculus refresh (limits, differentiation, partials)
    - Matrices & vectors (operations, rank, inverse), systems of linear equations
    - Intro optimization intuition (unconstrained minima)

??? note "Statistics I"
    - Descriptive measures, exploratory summaries
    - Probability axioms, counting, conditional probability & Bayes
    - Random variables (discrete, continuous), expectation & variance
    - Core distributions: Bernoulli, Binomial, Poisson, Normal, Exponential

### Core Data Science / Math
??? note "Linear Algebra"
    - Vector spaces, subspaces, span, basis, dimension
    - Linear independence, rank-nullity, orthogonality
    - Eigenvalues / eigenvectors, diagonalization, SVD intuition
    - Projections & least squares

??? note "Probability & Statistics II"
    - Joint / marginal / conditional distributions
    - Transformation of variables, Jacobians (intro)
    - Sampling distributions, LLN / CLT intuition
    - Estimation: bias, MSE, efficiency; confidence intervals
    - Hypothesis testing framework, Type I/II, power

??? note "Machine Learning Foundations"
    - Supervised learning pipeline & bias/variance trade-off
    - Regression (linear, regularization: Ridge/Lasso)
    - Classification (kNN, logistic regression, tree fundamentals)
    - Model evaluation: cross-validation, metrics (accuracy, precision/recall, ROC AUC)
    - Overfitting control (regularization, early stopping concept)

### Applied / Systems Layer
??? note "Databases & SQL"
    - Relational model, schema design (normal forms intro)
    - Joins, aggregations, indexing basics
    - Transactions & ACID (conceptual)

??? note "Business Data Management / Tools for DS"
    - Data cleaning pipelines, reproducible notebooks
    - Version control basics, environment management
    - Basic dashboards / reporting patterns
