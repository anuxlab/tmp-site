# Gauhati University – B.Tech in Computer Science & Engineering

**Type:** Undergraduate Engineering Degree  
**Period:** 2012 – 2016

**Foundations Built:** Data structures, algorithms, operating systems, databases, computer networks, software engineering fundamentals.

**Relevance:** Established core computational thinking & systems fundamentals later applied to data / quant engineering and automation work.

## Syllabus Overview (Condensed)
> Representative canonical CS topics (core + typical electives). Exact term ordering simplified.

### Programming & Data Structures
??? note "Intro Programming"
    - C programming basics, memory model, pointers
    - Modular programming, debugging patterns

??? note "Data Structures"
    - Arrays, linked lists, stacks, queues, hash tables
    - Trees (BST, AVL concepts), heaps, graphs representations
    - Complexity analysis & trade-offs

### Algorithms
??? note "Core Algorithms"
    - Sorting (quick, merge, heap), searching, recursion patterns
    - Graph algorithms: BFS, DFS, shortest paths (Dijkstra), MST (Kruskal/Prim)
    - Greedy vs dynamic programming paradigms

### Systems & Architecture
??? note "Computer Organization"
    - Number systems, digital logic basics
    - CPU architecture basics: ALU, control unit, registers, pipeline concept

??? note "Operating Systems"
    - Process vs thread, scheduling, synchronization primitives
    - Memory management: paging, segmentation
    - File systems & basic I/O

### Databases
??? note "DBMS"
    - ER modeling, relational algebra, schema normalization
    - SQL querying, transaction concepts, indexing

### Networks
??? note "Computer Networks"
    - Layered models (OSI/TCP/IP), addressing, routing basics
    - Transport protocols (TCP vs UDP), congestion control idea
    - Application layer examples (HTTP, DNS)

### Software Engineering
??? note "SE Principles"
    - SDLC models, requirements, version control introduction
    - Testing strategies (unit, integration)

### Theory Foundations
??? note "Discrete Mathematics"
    - Logic, sets, combinatorics, relations, functions
    - Graph theory basics

??? note "Automata / Computation (representative)"
    - Finite automata, regular expressions
    - Context free grammar concept

### Elective / Supplementary (Indicative)
- Intro Machine Learning (basics of classification/regression)
- Web Technologies (HTTP, HTML/CSS/JS fundamentals)

## Integration Forward
These fundamentals underpin data structure choices, performance reasoning, and reliability design in current market data & research pipeline engineering.
